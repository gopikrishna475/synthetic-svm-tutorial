# Synthetic Dataset with Support Vector Machine (SVM)

## Overview
This repository demonstrates how to apply a Support Vector Machine (SVM) classifier to synthetic random data.  
The focus is on understanding how kernel choice and hyperparameters affect classification performance.

## Contents
- `tutorial.html` – Webpage tutorial (<2000 words).
- `notebook.ipynb` – Jupyter notebook with complete code.
- `figures/` – PNG plots (confusion matrix, ROC curve, precision-recall).
- `requirements.txt` – Python dependencies.

## How to Run
```bash
git clone https://github.com/yourusername/synthetic-svm-tutorial.git
cd synthetic-svm-tutorial
pip install -r requirements.txt
jupyter notebook notebook.ipynb
